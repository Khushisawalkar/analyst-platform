# 📊 AnalystHub — Financial Analyst Discovery Platform

A modern web platform that connects retail investors with analysts by providing deep insights into analyst performance, financial data, and investment calls.

---

## 🌐 Live Demo

https://analyst-platform.vercel.app

---

## 🚀 Features

* 🔍 Search and filter analysts by niche
* 📈 Financial data visualization (Revenue, Net Income, EPS, Cash Flow)
* 🧠 CDI Score (Credibility Index) for analyst evaluation
* 📊 Peer comparison using scatter plots
* 📋 Recent calls and announcements
* ⚖️ Bull vs Bear analysis (pros & cons)
* ⭐ Follow analysts (UI-level state)
* 🎯 Clean, responsive dark UI

---

## 🛠 Tech Stack

* **Frontend:** React.js (Vite)
* **Visualization:** Recharts
* **Language:** JavaScript (ES6+)
* **Styling:** Custom CSS
* **Deployment:** Vercel
* **Version Control:** Git & GitHub

## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/Khushisawalkar/analyst-platform.git
cd analyst-platform
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
